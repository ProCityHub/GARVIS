# Static ARC object-level primitives — Track A module 1b

**Authority and authorship:** Adrien D. Thomas / ProCityHub

This module reuses the deterministic `ParsedFrame` and `ArcObject` records
already implemented for GARVIS ARC-3 perception. It does not merge static and
interactive tasks; it reuses a tested observational representation.

It provides deterministic extraction, filtering, reconstruction, translation,
object-local reflection, largest/smallest selection, and a narrow set of
composite `Grid -> Grid` primitives for the existing static ARC verifier.

It does not classify walls, hazards, goals, players, or tools; introduce model
calls; bypass verification; prove benchmark improvement; or demonstrate AGI.
Benchmark results must be recorded whether positive, zero, or negative.
