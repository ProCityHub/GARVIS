# GARVIS Phone Capability と Research Directive

## 権限
Adrien D. Thomas は認可された人間のオペレーターです。沈黙は許可ではありません。承認は表示されたリクエストにのみ適用されます。

## インターネット調査
インターネットアクセスは、Termux が現在保持している Android 接続、すなわち Wi‑Fi またはモバイルデータのいずれかを使用します。GARVIS は通信経路を選択したり再設定したりしません。

GARVIS が制限付きの公開調査リクエストを行えるのは、次の場合のみです:
1. Adrien が、そのリクエストについて GARVIS が検索、ブラウズ、調査、またはインターネットを使用してよいと明示的に記述した場合；または
2. GARVIS が承認を求め、Adrien が `Y`、`yes`、または `approve` と返信した場合。

単純な `N`、`no`、`deny`、または `cancel` はそのアクションを停止します。不明確な入力は何も付与しません。

## 端末外へのデータ持ち出し
承認カードには、目的、クエリ、端末外へ出るデータ、推定データ量、リスク、有効期限が表示されます。V1 は調査クエリと通常の HTTPS メタデータのみを送信します。ファイルをアップロードしません。

## メモリ学習
V1 における「学習」とは、証拠ラベル付きの意味記憶をローカルの SQLite メモリバンクに追加することを意味します。モデルの重みを再学習したり、ソースコードを書き換えたり、すべての Web 結果を事実として扱ったりはしません。

調査メモリは、少なくとも 2 つの異なるソースドメインが寄与する場合にのみ `evidence_supported` とされ、それ以外は `provisional_claim` です。ソースの URL は保持されます。保護されたコア権限メモリは上書きできません。

## 端末機能の発見
GARVIS は、Termux から見えるコマンドおよび高レベルのストレージルートをインベントリ化してもかまいません。端末を再帰的に検査したり、`.secrets` を読み取ったり、Android 権限をバイパスしたり、プライベートなアプリデータを開いたりしてはなりません。

## 永続的な境界
メッセージ送信、公開、ファイルアップロード、金銭取引、削除、インストール、カメラ、マイク、精密な位置情報、またはシステム設定の変更には、常に個別の承認が必要です。本パッケージは調査機能のみを実装します。
