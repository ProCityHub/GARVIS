# GARVIS アトラクタメモリ プロトタイプ

著者: Adrien D. Thomas

ステータス: 決定的なソフトウェア実験

科学的境界: 本プロトタイプは、意識、感性、同一性の持続、真の AGI、生物学的同等性、量子的検証、または物理法則を確立するものではありません。

## 目的

Hopfield 風の連想メモリコンポーネントを構築します。要件は次のとおりです。

1. 双極パターンを分散相互作用重みに格納する。
2. 完全・部分・ノイズを含む手がかりを受け付ける。
3. 反復的に安定状態へ収束する。
4. 収束、復元精度、エネルギー、失敗を報告する。

ネットワークの重みが記憶のメカニズムです。ストレージなしに記憶が存在すると表現してはなりません。

## 状態表現

格納されるパターンは -1 と +1 のみを含みます。

想起の手がかりは 0 を含んでもよく、これは消去または未知の値を表します。

すべてのパターンは同じ長さである必要があります。

## 重みの構築

次元 n の格納パターン x_k に対して:

W = (1 / n) * sum of outer_product(x_k, x_k)

重み行列は対称でなければなりません。

対角要素はすべて 0 でなければなりません。

想起では、ファイル名、データベースキー、パターン識別子、ルックアップインデックスではなく、重み行列を使用しなければなりません。

## 収束規則

決定的な非同期ニューロン更新を使用します。

各ニューロンについて:

field = sum(W[i][j] * state[j])

field が正ならニューロンを +1 に設定します。
field が負ならニューロンを -1 に設定します。
field が 0 の場合は現在の値を保持します。
field と現在値の両方が 0 の場合は +1 を使用します。

1 回の sweep はすべてのニューロンを 1 度ずつ更新します。

完全な 1 回の sweep で変更が発生しなかったとき、ネットワークは収束します。

有限の最大 sweep 回数は必須です。

## エネルギー

完全に双極の状態に対して:

E = -0.5 * state_transpose * W * state

決定的な非同期収束の間、エネルギーは増加すべきではありません。

これはソフトウェアの不変条件であり、物理的エネルギーの証拠ではありません。

## 必須測定項目

各想起結果は次を報告しなければなりません。

- 最終状態;
- 収束した/収束していない;
- sweep 回数;
- 状態変更の総数;
- エネルギーの推移;
- ターゲットが与えられた場合の完全一致可否;
- ターゲットが与えられた場合のハミング距離 (Hamming distance);
- 警告または失敗理由。

## 必須テスト

実装は次をテストしなければなりません。

1. 空のパターンの拒否。
2. 次元不一致の拒否。
3. 無効値の拒否。
4. 重みの対称性とゼロ対角。
5. 完全な格納手がかりからの完全想起。
6. 部分的な手がかりからの想起。
7. 制御されたビット反転ノイズからの想起。
8. 同一 seed 使用時の決定的結果。
9. 上限付きの非収束ハンドリング。
10. 非同期更新におけるエネルギー非増加。
11. 偽アトラクタの報告。
12. 複数パターンでの容量と干渉。

成功・失敗・曖昧な想起はすべて可視のままにする必要があります。

## ベースライン

実験は、アトラクタ想起を次と比較しなければなりません。

1. 明示的な最近傍パターンのルックアップ ベースライン。
2. 学習された連想的な収束を用いず、消去ビットを決定的に補完するメモリなしベースライン。

別の実験が事前登録されていない限り、比較は記述的です。

## エンジニアリング境界

最初の実装は Python 標準ライブラリのみを使用します。

提案モジュール:

src/garvis/attractor_memory.py

提案テスト:

tests/garvis/test_attractor_memory.py

メモリモジュールが行ってはならないこと:

- ファイルシステムへの書き込み;
- ネットワークアクセス;
- サブプロセッサーの実行;
- Git 操作;
- シークレットへのアクセス;
- モデル呼び出し;
- 外部アクション。

## 将来の GARVIS 統合

後の統合では、アトラクタの出力をオプションのコンテキストとして提供する場合があります。

想起された内容には出所と信頼度を含め、無批判の真実として扱ってはなりません。

アトラクタメモリは、セッション履歴、リポジトリの根拠、科学的主張、承認、外部アクション権限とは分離されたままにします。
