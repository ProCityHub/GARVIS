# GARVIS アトラクタ記憶プロトタイプ

著者: Adrien D. Thomas

ステータス: 決定的なソフトウェア実験

科学的な境界: 本プロトタイプは、意識、感受性、同一性の持続、真の AGI、生物学的同等性、量子的検証、または物理法則を確立しません。

## 目的

ホップフィールド型 連想メモリ コンポーネントを構築します。要件は次のとおりです。

1. 分散した相互作用重みに二値（双極）パターンを格納する。
2. 完全、部分、またはノイズを含むキューを受け付ける。
3. 反復的に安定状態へ収束する。
4. 収束、復元精度、エネルギー、および失敗を報告する。

ネットワークの重みが記憶の格納機構です。ストレージなしに記憶が存在すると記述してはなりません。

## 状態表現

格納されるパターンは -1 と +1 のみを含みます。

想起キューは 0 を含んでもよく、これは消去または不明な値を表します。

すべてのパターンは同じ長さでなければなりません。

## 重みの構築

次元が n の格納パターン x_k に対して:

W = (1 / n) * sum of outer_product(x_k, x_k)

重み行列 W は対称でなければなりません。

すべての対角成分は 0 でなければなりません。

想起は、ファイル名、データベースキー、パターン識別子、またはルックアップインデックスではなく、重み行列を使用しなければなりません。

## セトリング規則

決定的な非同期ニューロン更新を使用します。

各ニューロンについて:

field = sum(W[i][j] * state[j])

field が正ならニューロンを +1 に設定します。
field が負ならニューロンを -1 に設定します。
field が 0 のときは現在値を保持します。
field と現在値がともに 0 のときは +1 を使用します。

1 回のスイープで全ニューロンを 1 度ずつ更新します。

完全な 1 スイープで変更が 0 のとき、ネットワークは収束したとみなします。

有限の最大スイープ回数は必須です。

## エネルギー

完全に双極な状態に対して:

E = -0.5 * state_transpose * W * state

決定的な非同期セトリング中、エネルギーは増加してはなりません。

これはソフトウェア上の不変条件であり、物理エネルギーの証拠ではありません。

## 必須測定項目

各想起結果は次を報告しなければなりません:

- 最終状態;
- 収束／非収束;
- スイープ回数;
- 総状態変更数;
- エネルギーの推移;
- 目標が与えられた場合の厳密一致;
- 目標が与えられた場合のハミング距離;
- 警告または失敗理由。

## 必須テスト

実装は次をテストしなければなりません:

1. 空パターンの拒否。
2. 次元不一致の拒否。
3. 無効値の拒否。
4. 対称な重みとゼロ対角。
5. 完全な格納キューからの厳密想起。
6. 部分キューからの想起。
7. 制御されたビット反転ノイズからの想起。
8. 同一シードによる決定的な結果。
9. 有界な非収束時のハンドリング。
10. 非同期におけるエネルギー非増加。
11. 誤ったアトラクタの報告。
12. 複数パターンにおける容量と干渉。

成功、失敗、曖昧な想起はいずれも可視のままでなければなりません。

## ベースライン

本実験はアトラクタ想起を次と比較しなければなりません:

1. 明示的な最近傍パターン検索のベースライン。
2. 学習による連想的セトリングを用いずに、消去ビットを決定的に補完するメモリなしベースライン。

比較は、別個の実験が事前登録されていない限り、記述的なものに留まります。

## エンジニアリング境界

最初の実装は Python 標準ライブラリのみを使用します。

提案モジュール:

src/garvis/attractor_memory.py

提案テスト:

tests/garvis/test_attractor_memory.py

メモリモジュールが行ってはならないこと:

- ファイルシステムへの書き込み;
- ネットワークアクセス;
- サブプロセスの実行;
- Git 操作;
- シークレットへのアクセス;
- モデル呼び出し;
- 外部アクション。

## 将来の GARVIS 統合

後続の統合では、アトラクタ出力を任意のコンテキストとして提供する場合があります。

想起された内容には出所と信頼度を含める必要があり、決して無批判の真実として扱ってはなりません。

アトラクタ記憶は、セッション履歴、リポジトリの証拠、科学的主張、承認、および外部アクションの権限とは分離されたままにします。
