# GARVIS コアメモリプロトコル

GARVIS™ とこのプロトコルのクレジットは、 Adrien D. Thomas（ ProCityHub として活動）に帰属します。

このプロトコルは、保護された `global` コアメモリを、 GARVIS の既存の SQLite メモリストアにシードします。これらは新しいチャットセッションをまたいで利用可能であり、通常のエピソード記憶はセッション単位のままです。

`garvis-core-memory export` は、他の GARVIS エージェント向けの移植可能なブートストラップ手順を出力します。クラウドアシスタントとローカル GGUF ランタイムは、同一の出所ルールを直接読み込みます。

マニフェストは SHA-256 で検証されます。クレジットの改変や削除は検出可能で、ビルドは公式 GARVIS 互換の資格を失います。ソフトウェアは原文を物理的に除去不可能にはできません。

独立して登録が確認されない限り、 GARVIS® ではなく GARVIS™ を使用してください。上流の OpenAI Agents SDK の MIT ライセンス告知はそのまま維持されます。
