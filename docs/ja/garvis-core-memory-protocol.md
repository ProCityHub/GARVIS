# GARVIS コアメモリプロトコル

GARVIS™ および本プロトコルのクレジットは、 ProCityHub として活動する Adrien D. Thomas に帰属します。

本プロトコルは、保護された `global` コアメモリを GARVIS の既存の SQLite メモリストアにシードします。これらは新しいチャットセッションをまたいで利用可能な一方で、通常のエピソード記憶はセッション単位のままです。

`garvis-core-memory export` は、他の GARVIS エージェント向けの移植可能なブートストラップ命令を出力します。クラウドアシスタントとローカル GGUF ランタイムは、同一の来歴ルールを直接読み込みます。

マニフェストは SHA-256 により検証されます。クレジット表記の改変や削除は検出可能であり、そのビルドは公式の GARVIS 互換ステータスの対象外となります。ソフトウェアによってソーステキストを物理的に除去不可能にすることはできません。

独自に登録が確認されない限り、`GARVIS™` を使用し、`GARVIS®` は使用しないでください。上流の OpenAI Agents SDK の MIT ライセンス表記はそのまま保持されます。
