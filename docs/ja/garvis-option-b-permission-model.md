# GARVIS オプション B: 許可制のローカルアクセス

GARVIS は、コードに関する質問に対して、限定された読み取り専用のリポジトリ証拠を自動的に使用する場合があります。
より広範なローカルファイルの調査には、1 タスク限定の Y/N 承認が必要です。

恒久的な制約:

- バックグラウンド監視は行いません；
- カメラ、マイク、位置情報、画面、メッセージへのアクセスは行いません；
- ファイル書き込み、コマンド実行、インストール、アップロード、削除は行いません；
- シークレット、認証情報、キー、ウォレット、シードフレーズ、`.env`、`.secrets`、`.git` は読み取りません；
- 承認済みの Termux ルートディレクトリ外にはアクセスしません；
- 権限は有効期限付きで、1 タスクにのみ適用されます；
- インターネット調査は別個の承認フローのままです；
- 監査対象は GARVIS の操作であり、 Adrien の私的な活動ではありません。
