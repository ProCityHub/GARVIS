# GARVIS オプション B: 権限ゲート付きローカルアクセス

GARVIS は、コードに関する質問に対して、範囲を限定した読み取り専用のリポジトリの証拠を自動で使用する場合があります。
より広範なローカルファイルの検査には、1 タスク限定の Y/N 承認が必要です。

恒久的な制限:

- バックグラウンドでの監視は行いません;
- カメラ、マイク、位置情報、画面、メッセージへのアクセスは行いません;
- ファイル書き込み、コマンド実行、インストール、アップロード、削除は行いません;
- シークレット、認証情報、鍵、ウォレット、シードフレーズ、`.env`、`.secrets`、`.git` の読み取りは行いません;
- 承認された Termux ルート外へのアクセスは行いません;
- 権限には有効期限があり、1 つのタスクにのみ適用されます;
- インターネットでの調査は、引き続き別の承認経路です;
- 監査対象は GARVIS のアクションであり、 Adrien の私的な活動ではありません。
