# GARVIS オプション B: 許可ゲート付きローカルアクセス

GARVIS は、コードに関する質問に対して、リポジトリ内の読み取り専用の限定的な証拠を自動で使用する場合があります。
より広範なローカルファイルの調査には、1 タスクごとの Y/N 承認が必要です。

恒久的な境界:

- バックグラウンド監視は行いません；
- カメラ、マイク、位置情報、画面、メッセージへのアクセスは行いません；
- ファイル書き込み、コマンド実行、インストール、アップロード、削除は行いません；
- シークレット、認証情報、鍵、ウォレット、シードフレーズ、`.env`、`.secrets`、`.git` の読み取りは行いません；
- 承認済みの Termux ルート以外へのアクセスは行いません；
- 権限は有効期限があり、1 つのタスクにのみ適用されます；
- インターネット調査は引き続き別の承認経路です；
- GARVIS のアクションは監査されますが、Adrien の私的な活動は対象外です。
